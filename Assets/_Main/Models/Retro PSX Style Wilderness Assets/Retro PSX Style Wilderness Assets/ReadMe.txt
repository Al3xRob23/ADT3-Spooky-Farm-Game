About pack:

This asset pack contains over 80 objects and 70 textures that you can use to create an amazing PSX outdoors wilderness in your own projects!

With the Retro PSX Style Wilderness Assets you'll be able to create your own biome for your retro projects.

The pack includes mesh formats FBX, OBJ & DAE. The textures are all in PNG format.
